# Solana Token Validator - 仕様書 (Spec Sheet)

## 1. プロジェクト概要
Solanaブロックチェーン上のトークン（SPLトークン）の安全性を、DEXScreenerの市場データとGoPlus Securityの技術診断を組み合わせて多角的に検証するWebアプリケーション。

## 2. コア機能
1.  **CA/URL検索機能**: SolanaのCA、またはDEXScreenerのURLからCAを自動抽出して解析。
2.  **リアルタイム診断**: 最新の市場データ（流動性、時価総額）とセキュリティリスク（ハニーポット、権限設定）を取得。
3.  **コミュニティ熱量可視化**: 公式リンクの有無に加え、DEXScreener Boostの広告ポイントを表示。
4.  **7段階プロ・グレーディング**: 技術リスクとコミュニティ実体を組み合わせた高度な格付けシステム。
5.  **モダンUI/UX**: 省略表記と全桁表示の使い分け、垂直配置された主要メトリクス、レスポンシブなカードデザイン。

## 3. 判定アルゴリズム (7-Level Grading)

技術スコア（Base 100）に市場・運営要素を加味し、以下の7段階で評価します。

| ランク | 名称 | 判定基準 | 投資判断目安 |
| :--- | :--- | :--- | :--- |
| **🥇 S** | **Premium** | スコア90+ / 重大リスクなし / SNS・Webあり / **LP Burn済み** / **Boostあり** | 最高評価。安全性とプロモーションが完備。 |
| **🥈 A** | **Authenticity** | スコア85+ / 重大リスクなし / SNS・Webあり | 優良プロジェクト。広告(Boost)がないのみ。 |
| **🥉 B** | **Barebones** | スコア85+ / 重大リスクなし / **SNSまたはWebサイトが欠損** | 健全だがコミュニティ実態が不明な「骨組み」状態。 |
| **🔸 C** | **Caution** | スコア50~84。軽微な懸念あり（流動性不足、LP未焼却など）。 | 注意が必要な投機対象。 |
| **☠️ F+** | **Suspicious** | スコア30~49。重大な懸念（Mint権限有効など）またはリスクの累積。 | 非常に危険。運営による操作リスク大。 |
| **🔴 F** | **Fail** | スコア1~29。多くのレッドフラッグを検出。 | 投資非推奨。 |
| **💀 F-** | **Deadly** | **ハニーポット検出**、またはスコア0。 | **即死案件**。1円も入れるべきではない。 |

### スコア計算詳細
- **Honeypot**: 検出時 -100点 (即 F- 判定)
- **Mint Authority**: 有効時 -50点
- **Freeze Authority**: 有効時 -40点
- **LP Burned**: 95%以上焼却で +10点 / 未焼却時 -10点
- **Liquidity**: $50,000未満時 -20点
- **Web/SNS Links**: 各欠損につき -5点
- **DEXScreener Boosts**: 有効時 +5点
- **市場データ未検出**: -30点

## 4. UI/UX 仕様
- **ヘッダー**: トークンロゴ、名称、シンボル。
- **運営情報**: `🔥 BOOSTED (点数)` バッジと公式リンクボタンを表示。
- **市場メトリクス**: 時価総額・流動性を「$3.9M」のように短縮表示し、下に「$3,952,642」と詳細を追記。
- **垂直メトリクス**: スコア、24h価格変化、取引数（Buy/Sell）をカード右側に整理。
- **履歴管理**: 解析結果をカード形式で保持し、不要な結果は個別削除可能。

## 5. 技術スタック
- **Frontend**: Streamlit (Python)
- **APIs**:
  - DEXScreener API (Token & Pairs Data, Boosts)
  - GoPlus Security API (Solana Token Security)
- **Deployment**: Streamlit Cloud / Docker (想定)

# 🛡️ Solana Token Validator Pro

Solana上のミームコインのコントラクトアドレス（CA）またはDEXScreenerのURLを入力するだけで、市場データとセキュリティリスクを即座に解析し、投資判断をサポートする強力なバリデーターツールです。

## 🌟 主な機能

- **CA/URL自動解析**: コントラクトアドレスだけでなく、DEXScreenerのペアURLやトークンURLから自動的にCAを抽出します。
- **セキュリティ自動診断 (GoPlus API)**:
  - **ハニーポット判定**: 売却不可能などの致命的なリスクを検出。
  - **Mint/Freeze権限**: 追加発行や取引停止のリスクをチェック。
  - **LP Burn判定**: 流動性プールが焼却されているか（Rugpull防止）を検証。
- **市場データ解析 (DEXScreener API)**:
  - 時価総額（FDV）、総流動性、価格変化率（24h）、売買件数を一画面に表示。
  - 複数の取引ペアから最も流動性の高いものを自動選定。
- **高度なユーザー体験 (UX)**:
  - **履歴管理**: 解析したトークンを履歴として保持し、新しい順に並べて比較可能。
  - **自動クリア**: 検索入力後にフォームを自動クリアし、連続した解析をスムーズに行えます。
  - **個別削除**: 不要になった解析結果はXボタンで簡単に消去可能。
  - **ロゴ画像表示**: トークンのアイコンをヘッダーに表示し、対象を直感的に識別。

## 🛠 技術スタック

- **言語**: Python 3.10+
- **UIフレームワーク**: [Streamlit](https://streamlit.io/)
- **API連携**:
    - [DEXScreener API](https://dexscreener.com/developer/embed) (市場データ)
    - [GoPlus Security API](https://docs.gopluslabs.io/) (セキュリティデータ)

## 🚀 セットアップと実行方法

### 1. リポジトリのクローン
```bash
git clone https://github.com/agenda23/solana-token-validator-web-app.git
cd solana-token-validator-web-app
```

### 2. 依存ライブラリのインストール
```bash
pip install -r requirements.txt
```

### 3. アプリの起動
```bash
streamlit run app.py
```

起動後、ブラウザで `http://localhost:8501` にアクセスしてください。

## 📊 判定ランクについて

安全性と流動性を独自のアルゴリズムでスコアリングし、以下の3段階で評価します。

- **🟢 S ランク**: 重大なリスクなし。流動性が高く、LPが焼却されている優良案件。
- **🟡 B ランク**: 注意が必要。一部の権限設定が有効であるか、流動性がやや低い状態。
- **🔴 F ランク**: 危険。ハニーポット検出、致命的な権限設定、または極めて低い流動性。
